# 🌍 GoOutside Market Performance Dashboard

## Overview

GoOutside is a growing outdoor retail company operating across multiple markets. This project delivers an interactive Business Intelligence solution that enables stakeholders to monitor sales performance, compare market trends, and identify revenue-driving products.

The solution transforms raw business data into actionable insights through SQL-based data transformation, cloud data warehousing in BigQuery, and interactive visualization in Data Studio (formerly Looker Studio).

## 🎯 Business Objectives

| BigQuery      | Cloud data warehouse                        |
|Google Sheets  | Data connection and reporting layer         |
| Data Studio | Dashboard development and reporting         |

## 📈 Key Insights

* Market-level performance comparison across regions
* Product category analysis and top-selling products
* Revenue trends over time
* Interactive filtering for deeper business exploration
* Executive KPI monitoring through a centralized dashboard

## 🚀 Project Workflow

Raw Data → SQL Transformation → BigQuery Data Warehouse → Google Sheets → Data Studio Dashboard → Business Insights
